Blue Mountain Springs

Documentation

springs

The top level of the service. The list of all available resources.

springs/magazines

The set of magazines in Blue Mountain.

example
restxq/springs/magazines

springs/magazines/{bmtnid}

The magazine(s) with the id bmtnid.

example
restxq/springs/magazines/bmtnaap

springs/magazines/{bmtnid}/issues

The issues of the magazine(s) with the id bmtnid.

example
restxq/springs/magazines/bmtnaap/issues

springs/constituent/{$issueid}/{$constid}

A specific constituent of a magazine issue.

Engages content negotiation.

  • text/plain
  • application/tei+xml

Try it:

  • curl -H"Accept:text/plain" http://HOST/restxq/springs/constituent/bmtnaap_1921-11_01/c003
  • curl -H"Accept:application/tei+xml" http://HOST/restxq/springs/constituent/bmtnaap_1921-11_01/c003

springs/constituents?byline={string}

A set of constituents whose byline includes string.

application/tei+xml
Returns a teiCorpus containing TEI documents, one per constituent. the teiHeader of each constituent contains bibliographic info on the constituent.

Try it:

  • curl -H"Accept:application/tei+xml" http://HOST/restxq/springs/constituents?byline=Tzara
